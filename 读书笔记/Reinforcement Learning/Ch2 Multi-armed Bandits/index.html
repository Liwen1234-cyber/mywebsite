1. A kkk-armed Bandit Problem​

重复地在 kkk 个不同的选项或者行为中进行选择，每次选择后，从一个固定的概率分布中收获一个数字的奖励。

你的目标是，最大化一定时段内所有收益和的期望。

We deonte action selected on time step ttt as AtA_tAt​， and the corresponding reward as RtR_tRt​, the value of an arbitrary action aaa denoted q∗(a)q_*(a)q∗​(a) as

q∗(a)≐E[Rt∣At=a]q_*(a) \doteq \mathbb{E}[R_t|A_t = a]q∗​(a)≐E[Rt​∣At​=a]

我们假设，你不知道每一个行为的确切价值，但你可能有个估计，记为　Qt(a)Q_t(a)Qt​(a)。

  • Exploiting: 当贪心地根据当前知识选择最大值的行为。可能在当前的１步中可以最大化
  • Exploring: 选择非贪心的行为，可能在长期的收益总和中获得最大值

在单次的选择中，不可能同时进行explore和exploit。 因此，平衡这两个操作是非常关键的。

2. Action-value Methods​

A natural way to estimate the value is by averaging the rewards actually received:

Qt(a)≐∑i=1t−1Ri⋅1Ai=a∑i=1t−11Ai=aQ_t(a) \doteq \frac{\sum_{i=1}^{t-1}R_i\cdot\mathbb{1}_{A_i=a}}{\sum_{i=1}^{t-1}\mathbb{1}_{A_i=a}}Qt​(a)≐∑i=1t−1​1Ai​=a​∑i=1t−1​Ri​⋅1Ai​=a​​

采样平均（sample-average）估算方法：随着上式分母增大，根据大数定理，Qt(a)→q∗(a)Q_t(a) \to q_*(a)Qt​(a)→q∗​(a) 。

采用采样平均方法来进行行为选择：最简单的是greedy方法，选择有着最大期望值的行为：

At≐arg⁡amax⁡Qt(a)A_t \doteq \arg_a\max Q_t(a)At​≐arga​maxQt​(a)

贪婪行为选择总是会用尽当前的知识以最大化立得收益。

ϵ\epsilonϵ-greedy 方法：大多时间选择 greedy 方法，以 ϵ\epsilonϵ 的概率选择探索。

3. The 10-armed Testbed​

可以看到，greedy 方法比其他的方法要提升地略快一些，在最一开始的时候，但很快会停止更新。会陷入次优的行为。

ϵ\epsilonϵ 较小的方法会最终得到比较好的效果。

然而，ϵ\epsilonϵ-greedy 的优势要根据任务而定。

收益的变动越大，使用 ϵ\epsilonϵ-greedy 的效果越好。反之，使用 greedy 效果好。

Reinforcement learning requires a balance between exploration and exploitation.

5. Tracking a Nonstationary Problem​

平均方法对稳定bandit问题是比较适合的，但是对于非稳定的问题，如果给近期的收益更大一点的权重比较合理。

Qn+1≐Qn+α[Rn−Qn]Q_{n+1} \doteq Q_n + \alpha[R_n-Q_n]Qn+1​≐Qn​+α[Rn​−Qn​]

Qn+1=(1−α)nQ1+∑i=1nα(1−α)n−iRiQ_{n+1} = (1-\alpha)^nQ_1+\sum_{i=1}^n\alpha(1-\alpha)^{n-i}R_iQn+1​=(1−α)nQ1​+i=1∑n​α(1−α)n−iRi​

可以证明：

(1−α)n+∑i=1nα(1−α)n−i=1(1-\alpha)^n + \sum_{i=1}^n \alpha(1-\alpha)^{n-i} = 1(1−α)n+i=1∑n​α(1−α)n−i=1

变动的步长参数需要满足：

∑n=1∞αn(a)=∞ and ∑n=1∞αn2(a)<∞\sum_{n=1}^\infty \alpha_n(a) = \infty \text{ and } \sum_{n=1}^\infty \alpha_n^2(a) < \infty n=1∑∞​αn​(a)=∞ and n=1∑∞​αn2​(a)<∞

前一个是为了克服初值条件或者随机扰动，后者是为了确保收敛。

6. Optimistic Initial Values​

7. Upper-Confidence-Bound(UCB) Action Selection​

虽然 ϵ\epsilonϵ-greedy 方法可以尝试exploration，但是对于各个行为的选择没有偏好。比较理性的方法是尝试更有潜力的行为。

一种有效的非贪婪行为选择方法是：

Ay≐arg⁡amax⁡[Q(a)+cln⁡tNt(a)]A_y \doteq \arg_a\max\left[Q_(a)+c\sqrt{\frac {\ln t} {N_t(a)}} \right]Ay​≐arga​max[Q(​a)+cNt​(a)lnt​​]

其中，Nt(a)N_t(a)Nt​(a) 是行为 aaa 的计数器。置信系数c>0c>0c>0　控制exploration的程度。若　Nt(a)=0N_t(a)=0Nt​(a)=0，那么就选最大值的行为。

平方根用于控制不确定的程度。

但是，这个方法比ϵ\epsilonϵ-greedy 更为困难：

  1. 处理非稳定问题时，这个处理使得问题更为复杂
  2. 处理状态空间比较大的问题时，使用函数逼近的方法也更为困难。

因此，在更先进的方法中，这个UCB的想法常常会不切实际。

9. Associative Search (Contextual Bandits)​

目前，所涉及的问题仅仅是非关联（nonassociative）的任务，即，不同的行为不需要关联不同的环境情形。

比如对于 kkk-armed bandit 问题，每一步的概率都会变。

现在假设，你可以知道某些暗示，比如颜色和胜率的关系，那么你可以将之关联。

关联搜索任务：包含

  1. trial-and-error 学习以搜索最佳的行为
  2. 关联这些行为与最适合的执行情形。
