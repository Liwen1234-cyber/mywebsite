Gram 矩阵​

欧氏空间

V 是 R 上的线性空间，定义映射

σ:V×V→R

对于 α,β∈V，将 σ(α,β) 记为 ⟨α,β⟩，若 σ 满足：

  1. 对称性：⟨α.β⟩=⟨β,α⟩
  2. （右）齐次性：⟨α,kβ⟩=k⟨α,β⟩
  3. （右）可加性：⟨α,β+γ⟩=⟨α,β⟩+⟨α,γ⟩
  4. 非负性：⟨α,α⟩≥0，且 ⟨α,α⟩=0⇔α=0

则称 σ 为 V 上的（实）内积，当 V 是有限维时，称其为欧氏空间（Rn 为标准欧氏空间）

实际上 α 是一个向量，β 是一个向量，⟨α,β⟩ 表示向量 α 与向量 β 的内积，结果是一个实数

实内积的性质

  1. （左）齐次性：⟨kα,β⟩=k⟨α,β⟩
  2. （左）可加性：⟨α+β,γ⟩=⟨α,γ⟩+⟨β,γ⟩
  3. ⟨k1​α1​+⋅⋅⋅+ks​αs​,β⟩=k1​⟨α1​,β⟩+⋅⋅⋅ks​⟨αs​,β⟩
  4. ⟨α,k1​β1​+⋅⋅⋅+ks​βs​⟩=k1​⟨α,β1​⟩+⋅⋅⋅ks​⟨α,βs​⟩

复内积

V 是 C 上的线性空间，定义映射

σ:V×V→C

对于 α,β∈V，将 σ(α,β) 记为 ⟨α,β⟩，若 σ 满足：

  1. 共轭对称性：⟨α.β⟩=⟨β,α⟩​
  2. （右）齐次性：⟨α,kβ⟩=k⟨α,β⟩
  3. （右）可加性：⟨α,β+γ⟩=⟨α,β⟩+⟨α,γ⟩
  4. 非负性：⟨α,α⟩≥0，且 ⟨α,α⟩=0⇔α=0

则称 σ 为 V 上的（复）内积，当 V 是有限维时，称其为酉空间（Rn 为标准欧氏空间）

复内积的性质

  1. （左）齐次性：⟨kα,β⟩=kˉ⟨α,β⟩
  2. （左）可加性：⟨α+β,γ⟩=⟨α,γ⟩+⟨β,γ⟩
  3. ⟨k1​α1​+⋅⋅⋅+ks​αs​,β⟩=k1​​⟨α1​,β⟩+⋅⋅⋅ks​​⟨αs​,β⟩
  4. ⟨α,k1​β1​+⋅⋅⋅+ks​βs​⟩=k1​⟨α,β1​⟩+⋅⋅⋅ks​⟨α,βs​⟩

线性组合的内积的矩阵表示

α1​,...,αs​;β1​,...,βt​ 是 C 上的内积空间 V 中的两个向量组，则

⟨k1​α1​+⋅⋅⋅+ks​αs​,l1​β1​+⋅⋅⋅+lt​βt​⟩=(k1​​,...,ks​​)​⟨α1​,β1​⟩⋮⟨αs​,β1​⟩​⋯⋱⋯​⟨α1​,βt​⟩⋮⟨αs​,βt​⟩​​​l1​⋮lt​​​​

Gram 矩阵

α1​,...,αs​;β1​,...,βt​ 是 C 上的内积空间 V 中的两个向量组，则

​⟨α1​,β1​⟩⋮⟨αs​,β1​⟩​⋯⋱⋯​⟨α1​,βt​⟩⋮⟨αs​,βt​⟩​​

称为 α1​,...,αs​;β1​,...,βt​ 的协 Gram 矩阵，记为 G(α1​,...,αs​;β1​,...,βt​)

α1​,...,αs​ 是 C 上的内积空间 V 中的一个向量组，则

​⟨α1​,β1​⟩⋮⟨αs​,β1​⟩​⋯⋱⋯​⟨α1​,βt​⟩⋮⟨αs​,βt​⟩​​

称为 α1​,...,αs​ 的 Gram 矩阵，记为 G(α1​,...,αs​)

α1​,...,αs​ 是 Cn 中的一个向量组，记 A=(α1​,...,αs​)，则

G(α1​,...,αs​)=AHA

其中，AH=(Aˉ)T=(AT)​

α1​,...,αs​ 是 Rn 中的一个向量组，记 A=(α1​,...,αs​)，则

G(α1​,...,αs​)=ATA

α1​,...,αs​;β1​,...,βt​ 是 C 上的内积空间 V 中的两个向量组，如果 α1​,...,αs​ 可由 β1​,...,βt​ 线性表出，且

(α1​,...,αs​)=(β1​,...,βt​)A

则

G(α1​,...,αs​)=AHG(β1​,...,βt​)A

Gram 矩阵的性质

  1. rank(G)=rank(α1​,...,αs​)
  2. Hermite 性：GH=G
  3. 非负性：∀x∈Cs，复二次型 xHGx≥0，并且 G 正定 ⇔α1​,...,αs​ 线性无关
